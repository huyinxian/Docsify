# 指针

## 指针和引用

---

指针是一个变量，存储的是**地址**，而引用则是变量的**别名**。指针在运行时可以改变所指向的对象，而引用在初始化绑定对象后就不允许再更改。

如果从内存的角度来看，由于指针是变量，因此会分配内存，但引用时不会分配内存的。因此，引用声明时必须指向一个对象，且不能指向空值。

```cpp
int **p1;   // 合法，指向指针的指针
int *&p2;   // 合法，指向指针的引用
int &*p3;   // 不合法，指向引用的指针
int &&p4;   // 不合法，指向引用的引用
```

引用可以指向指针，但是指针不能够指向引用，因为引用本身只是变量的别名，并不是一个对象。
